## 💻 Rotas:

| Rotas        | HTTP   | Descrição                            |
| ------------ | ------ | ------------------------------------ |
| /:hash       | GET    | Redireciona para o Link Encurtado    |
| /short       | POST   | Criar Link Encurtado                 |
